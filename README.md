# PhotoStats
Statistics about photos


## DEV setup
1. Setup VENV
```
python3 -m venv .venv
```

2. Setup DJANGO

3. Configuration
3.1 Remove 'example_' from files in [config](.config/)
3.2 Maintain variables in there.


4. Run
```
$ cd photostats
$ python manage.py runserver
```
